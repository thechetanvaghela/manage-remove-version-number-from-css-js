This plugin provide an option to manage or remove the version number from CSS and JS files.

== Description ==

The plugin allows you to manage or remove the version number from CSS and JS files.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the "Manage/Remove version number from CSS & JS" plugin through the 'Plugins' screen in WordPress
3. Visit "Version No. of JS&CSS" from admin menu
